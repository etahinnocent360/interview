## if you don't have nestjs install on your computer please run the following command on the terminal
```
 $ npm i -g @nestjs/cli
 ```

## run the following command to install project dependecies

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
